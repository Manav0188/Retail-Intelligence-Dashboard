# Retail-Intelligence-Dashboard

## Description
End-to-end sales analysis using SQL and Power BI to derive actionable business insights from retail data.

## Project Overview
A comprehensive retail sales analysis project integrating SQL for data extraction and Power BI for interactive dashboarding. It identifies trends, seasonal variations, and key performance indicators to improve retail decision-making.

## Skills Demonstrated
- **Data Analysis:** Extracting insights from complex datasets.
- **Visualization:** Building interactive reports and dashboards.
- **Technical Tools:** Power-Bi, Sql, Sales-Analysis, Business-Intelligence, Data-Visualization.
